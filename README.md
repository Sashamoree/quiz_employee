# quiz_employee
#  Система тестирования знаний сотрудников

> Веб-приложение для проведения тестирования сотрудников с гибким управлением вопросами, детальной статистикой и формированием отчетов.

##  Особенности

###  Интерактивное тестирование
- **Пошаговый режим** — вопросы появляются по одному, что снижает нагрузку на пользователя
- **Прогресс-бар** — наглядное отображение хода тестирования
- **Случайная выборка** — каждый раз формируется уникальный набор из 15 вопросов (из имеющейся базы)
- **Автоматический подсчет** — моментальный расчет результата после завершения

###  Управление вопросами
- **Полный CRUD** — добавление, редактирование и удаление вопросов
- **Поиск и фильтрация** — быстрый поиск по тексту вопроса
- **Пагинация** — удобная навигация по списку вопросов
- **3 варианта ответа** — каждый вопрос может иметь 3 варианта с одним правильным

###  Расширенная статистика
- **Фильтрация данных** — по ФИО, отделу и диапазону дат
- **Карточки показателей** — общее количество тестов, успешно сдавших, процент успешности
- **Экспорт в Excel** — выгрузка отфильтрованных данных с сохранением форматирования
- **Детальный просмотр** — анализ ошибок каждого сотрудника с указанием правильных ответов

###  Формирование отчетов
- **Компактный дизайн** — отчет помещается на одном листе A4
- **Черно-белая печать** — оптимизировано для офисной техники
- **Все данные сотрудника** — ФИО, звание, отдел, экзаменатор
- **Таблица с ответами** — сравнение ответов сотрудника с правильными
- **Подписи** — предусмотрены поля для подписи экзаменатора и сотрудника

###  Учет экзаменаторов
- **Выбор из списка** — предустановленный список экзаменаторов
- **Звания** — возможность указать звание для сотрудника и экзаменатора
- **Отображение в отчетах** — все данные сохраняются и выводятся в отчете

##  Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/Sashamoree/quiz_employee
cd quiz_employee
```
### 2.Создаем виртуальное окружение
```bash
python3 -m venv venv
source venv/bin/activate #macOS / Linux
venv\Scripts\activate #Windows
```
### 3. Устанавливаем зависимости
```bash
pip install -r requirements.txt
```
### 4. Запуск приложения
```bash
python3 app.py
```
### 5. Доступ к приложению
```bash
http://127.0.0.1:5000
```
### 6. Добавляем экзаменаторов и должности тестируемых сотрудников
```python
nano data.py
EXAMINERS = [
    {'name': 'Иванов Иван Иванович', 'rank': 'Старший специалист отдела ИБ'},
    # добавьте своих значения
]
RANKS = [
    'Специалист',
    'Главный специалист',
    'Начальник отдела',
    # добавьте свои значения
]
```
### 7. Проходной бал
```python
nano app.py
```
